Squad 1 • Infrastructure

Resilient S3 CDN Hosting

A high-availability static assets deployment blueprint. Integrates Amazon Simple Storage Service (S3) bucket policies with CloudFront CDNs to guarantee sub-millisecond static latency globally. Fully secured with Let's Encrypt SSL certificates processed automatically via Route53 DNS validation parameters.

Skills Gained

  • CDN Edge Caching
  • ACM SSL Management
  • Bucket Policy Lockdown
  • DNS Failover Systems

Project Outcome

Designed and launched a secure, production-grade static asset pipeline with sub-100ms global response time and automated custom SSL certificate rotations.

Amazon S3 CloudFront CDN Route 53 ACM SSL
Squad 2 • Serverless

Serverless AI Video Summarizer

An automated AI pipeline triggered by file additions in media storage directories. Pushing an MP4 triggers AWS SQS/Lambda workers which feed transcripts into Bedrock endpoints, outputting summarized logs into DynamoDB database systems autonomously.

Skills Gained

  • Event-Driven Compute
  • AI Model Prompting
  • NoSQL Schema Design
  • Message Queue Decoupling

Project Outcome

Built a fully serverless, self-scaling media summaries architecture processing multi-gigabyte video files for a cost of under $0.01 per runtime hour.

AWS Lambda DynamoDB AWS Bedrock SQS Queues
Squad 3 • Containerization

Containerized ECS Task SaaS

A high-availability enterprise Docker task manager running across AWS Elastic Container Service (ECS) backed by serverless AWS Fargate nodes. Routes traffic using Application Load Balancers (ALB) and guarantees database safety using Multi-AZ RDS Postgres configurations.

Skills Gained

  • Docker Multi-stage Builds
  • Load Balancer Rules
  • Multi-AZ Failover
  • ECS Task Definitions

Project Outcome

Designed and deployed a modern three-tier containerized web app, featuring high tolerance against database node losses and load peak mitigation.

ECS Fargate AWS ALB RDS Postgres Docker

Multi-Region Disaster Recovery

  • Automated Health Failover
  • Replica Lag Tuning
  • Trans-Atlantic Routing

AWS Aurora Global Route 53 ARC Terraform IaC VPC Peering
