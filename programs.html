The Breakdown

Detailed Weekly Curriculum

Step-by-step weekly milestone breakdown engineered to guarantee successful, functional infrastructure creation.

Week 1

Foundations

Duration: 1 Week

Security Baseline & Command Terminals

Every enterprise deployment starts with a rigorous security standard. Week 1 is structured to ensure squad members configure AWS accounts securely. We establish IAM role configurations, generate SSH keys, install AWS CLI, and sync with shared git repositories.

AWS IAM MFA Local SSH Keys AWS CLI config GitHub Setup
Week 2

Architecture

Duration: 1 Week

VPC Mapping & Network Subnets

Design custom high-availability Virtual Private Clouds. Squads map out multi-availability-zone networks featuring public subnets for public ingress routers, isolated private subnets for application nodes, and network address translators (NAT).

Multi-AZ Subnets NAT Gateways Security Groups VPC Flow Logs
Week 3

Development

Duration: 1 Week

Application Packing & Containerization

Code server layers and package them inside secure container formats. Squads write Express Node.js servers, configure Docker environments, optimize files to reduce sizes, and push outputs to Amazon Elastic Container Registry (ECR).

Node.js API Dockerfiles Multi-stage build AWS ECR Registry
Week 4

Deployment

Duration: 1 Week

Infrastructure as Code (IaC) with Terraform

Automate cloud creations to prevent configuration drift. Cohort members write declarative Terraform files defining ECS target pools, S3 assets repositories, server compute parameters, and load balancers.

Terraform Modules S3 State Locks Plan & Apply AWS ECS Fargate
Week 5

Testing

Duration: 1 Week

Integration Workflows & Security Auditing

Ensure codebase changes run testing checks before deployment. Squads set up automated GitHub actions pipelines, configure secrets in environment vaults, execute security audits to discover secret leaks, and fix runtime errors.

GitHub Actions CI/CD variables Snyk Auditing Security Groups
Week 6

Demos

Duration: 1 Week

Architectural Showcase & Reviews

Present your working product structures! Cohort members explain systems, present code-bases, outline load tests results, and undergo reviews with technical advisors, culminating in accredited certificates.

Live Demo zoom Portfolio Host Accreditations
